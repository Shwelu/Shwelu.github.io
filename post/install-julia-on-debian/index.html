How to install Julia on Debian 8.5

Mon, Apr 17, 2017 julia, debian

Introduction

Introduced in 2012, Julia is an interactive language which can be used to quickly prototype a software solution as well as to write high performance code. Using Low Level Virtual Machine Just in Time (LLVM JIT) compiler technologies, Julia combines the expressive power of high level languages such as R or Python with the raw performance of statically typed and compiled languages such as C or FORTRAN.

This short tutorial will guide you how to install Julia on Debian.

Installation

Debian does not provide the latest Julia version in its default software repositories. Therefore default installation command sudo apt-get install may not work out of the box on some Linux distros such as Debian.

For Julia, there exists a community-maintained Personal Package Archive (PPA). PPA allows package developers to upload Ubuntu source packages to be built and published as an apt repository by Launchpad.

Before installing software-properties-common package, update the apt package index to download information about the new packages:

sudo apt-get update

Once the repository is updated, we’ll install the software-properties-common package:

sudo apt-get install software-properties-common

This command installs the Software Properties Common package that includes the add- apt-repository binary. Now you are ready to add the custom PPA. Add the repository for Julia and its dependencies:

sudo add-apt-repository ppa:staticfloat/juliareleases   
sudo add-apt-repository ppa:staticfloat/julia-deps

Enter your account password if prompted.
Now you are ready to install Julia:

sudo apt-get install julia
sudo apt-get update

The Julia should now be ready to use. You can check the version of Julia:

julia -v

Output will be similar to the following:

julia version 0.3.2

That’s it! Julia is installed on your server.
